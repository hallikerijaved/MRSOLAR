# MR Solar Landing Page

This project is a responsive single-page website for **MR Solar**, built with plain HTML and CSS.

## Project Files

- `index.html` - main website markup
- `css/style.css` - all styling and responsive layout rules
- `assets/` - logo and image assets used by the page

## Features

- Responsive landing page design
- Fixed navigation with mobile menu
- Hero section with call-to-action buttons
- Services, projects, process, and contact sections
- Contact form layout
- Embedded Google Map

## How To Run

1. Open the project folder.
2. Double-click `index.html`, or open it in any web browser.

## Editing

- Update text content in `index.html`
- Update styles and mobile responsiveness in `css/style.css`
- Replace logo or images inside the `assets/` folder as needed

## Notes

- The site uses Google Fonts through a CDN link.
- The layout is designed to work on desktop, tablet, and mobile screens.
